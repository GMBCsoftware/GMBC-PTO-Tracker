# Instructions
Attached in this folder are several csvs. These represent the various sheets attached to the google sheet "TimeOffApp" which the Google AppScripts client uses for storage

## Structure
- `Sheets/Employees.csv` - lists all employees in the sheet. Fake data is used for reference.
- `Sheets/Requests.csv` - List and header names of all requests made for time-off. Fake data added for refernce.
- `Sheets/Settings.csv` - List of settings for admin use.
- `Sheets/Audit.csv` - list of changes and stores error logs.
